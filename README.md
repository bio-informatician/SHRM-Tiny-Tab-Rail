# SHRM - Tiny Tab Rail Overlay

**Tiny Tab Rail Overlay** adds a vertical sidebar to your browser showing all tabs with icons only, pinned tabs on top, and hover preview. Fully synchronized across tabs.

**Source code & updates:** [GitHub Repository](https://github.com/bio-informatician/SHRM-Tiny-Tab-Rail)

## Features
- Persistent, browser-wide sidebar
- Pinned tabs appear at the top
- Hover tooltip with tab title
- Auto-hide collapsed bar (thin blue line)
- Right-click tab to pin/unpin
- Active tab highlight
- Fully page-safe and namespaced
